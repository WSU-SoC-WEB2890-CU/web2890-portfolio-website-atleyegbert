    Study Smart: Homework + Due Date Management

    Project Description

    Tools + Technology:

    JS React, Bootstrap,
    Responsive Design

    Go To Site

    The React app I developed is a task management app that allows a user to add classes and assignments, track their due dates, and remove them after they are completed. The target audience for this app was students who struggle to manage their workload and would like a way to track their progress through the semester better. The purpose of this app was to help these students to track assignments more easily and be able to stay on top of school work. I used Bootstrap to make sure my design looks good and scales for multiple screen sizes.

    One area that I struggled with making this app is that I wanted to use a database to store the information for the classes and assignments that users add. However, I had a lot of trouble with posting the submissions. As I develop this app in the future, I will definitely figure out a way to do this. Since I wanted to spend more time working on other feature of the app, I chose to add some preloaded data instead so that the app would still be able to function.

    React App Design

    • Home Page

      On the main page I included a list of all assignments, organized by their due date. This is a simple design that clearly outlines the assignment name, the class it is for, and the due date. This improves a user's experience because they do not have to search around the app to find the information that is most crucial to them.

    • Assignment Detail Page

      When an assignment is selected, it opens to a page that displays more information about that assignment, this includes any notes a student has written about it. This improved the user's experience because they are able to add optional information about an assignment without overcrowding the home page.

    • Class Schedule Page

      On the class schedule page, there is an option to add classes and view a list of all classes. This design is also straightforward so that it is not overwhelming to users. This improved their experience by allowing them to keep track of the classes they are taking.

    • Class Detail Page

      When a class is selected, users are able to view a page with the list of all the assignments due for that class and be able to add assignments. This adds to the user's experience because they are be able to view assignments based on the class they have selected.
      This page also has a form which allows them to add new assignments. The form is hidden until they press the button to add a new assignment. The form has inputs for the assignment name, due date, and any additional notes.
